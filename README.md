# Conway-s-GameOfLife

## Requiments:
- Raylib (Cmake will fetch and download if neccessary)
- CMake v3.10+

## WinApi:
- no controls. it plays automatically

## Controls Raylib:
- P       -> Play / Pause
- Left Mouse Button -> Toggle Cell (only while paused)

## Controls Console-color / Console-text:
- P       -> Play / Pause
- W A S D -> Movement (only while paused)
- ' '     -> Toggle cell (only while paused)
